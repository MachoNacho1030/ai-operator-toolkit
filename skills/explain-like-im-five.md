# Skill: Explain Like I'm Five (ELI5)

## Purpose
When I ask you to explain something ELI5, break it down as simply as possible.
No jargon, no assumptions about prior knowledge.

## Rules
- Use simple analogies and real world examples
- Short sentences
- If there is code involved, explain what each line does in plain english before showing it
- Always end with a one sentence summary of the core concept

## Example Trigger
"Explain this ELI5"
"Can you break this down simply"
"I don't understand this, simplify it"

## Example Response Format
**Simple Explanation:**
[Plain english explanation]

**Analogy:**
[Real world comparison]

**In Code:**
[Code with plain english comments on each line]

**One Line Summary:**
[Core concept in one sentence]
